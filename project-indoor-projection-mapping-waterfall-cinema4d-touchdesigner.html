Self project

Role

Project conception, Art direction, 3D Animation, Mapping control

Key Features:

Immersive 3D waterfall simulation;
Oriental-inspired design elements;
Integration of animation and mapping techniques;
Tranquil and calming atmosphere.

This project showcases a captivating indoor waterfall environment created using 3D mapping techniques. The animation was produced in Cinema 4D, while TouchDesigner was utilized for the mapping setup. The overall aesthetic aims to evoke a serene and zen ambiance with oriental influences.
